scikits.umfpack.UmfpackLU.perm_c¶

property UmfpackLU.perm_c¶

Permutation Pc represented as an array of indices.

The column permutation matrix can be reconstructed via:

>>> Pc = np.zeros((n, n))
>>> Pc[np.arange(n), perm_c] = 1
