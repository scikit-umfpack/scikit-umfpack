scikit-umfpack¶

scikit-umfpack provides a wrapper of UMFPACK sparse direct solver [1], [2], [3] to SciPy.

Usage:

>>> from scikits import umfpack
>>> lu = umfpack.splu(A)
>>> x = umfpack.spsolve(A, b)

Installing scikits.umfpack also enables using UMFPACK solver via some of the scipy.sparse.linalg functions, for SciPy >= 0.14.0.

You can obtain scikit-umfpack from

  • https://pypi.python.org/pypi/scikit-umfpack (releases)

  • https://github.com/scikit-umfpack/scikit-umfpack (development version)
